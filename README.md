# MBTI 学习 Prompt 库 · 16 型专属学习方式

> 不是你学不会，是学习方法没匹配你的思维习惯。
> 这个仓库开源了 16 份基于 MBTI 认知功能设计的学习 Prompt，任何 LLM 都能直接用。

## 快速开始

1. 打开 Claude / ChatGPT / Gemini
2. 进入 `prompts/` 找到你的 MBTI 类型文件
3. 复制全部内容作为 System Prompt
4. 把你要学的主题/材料粘进去，开始对话

## 设计原理

每份 Prompt 围绕 MBTI 的**认知功能堆栈**设计，而不是 4 字母标签，覆盖 6 个维度：

| 维度 | 说明 |
|---|---|
| 讲解结构 | 先全局还是先细节？线性还是网状？ |
| 抽象层次 | 偏原理还是偏案例？ |
| 节奏控制 | 密集推进还是留白探索？ |
| 反馈语气 | 直接纠错还是先肯定？ |
| 举例方式 | 数据/故事/类比/场景 |
| 激励机制 | 挑战感/意义感/秩序感/体验感 |
